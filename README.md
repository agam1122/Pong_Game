# Pong_Game


-Pong Game-



Description:-

Welcome to the Pong Game! This simple two-player game allows you to control paddles using different keys. Player 1 controls their paddle with 'W' and 'S' keys, while Player 2 uses the up and down arrow keys. The objective is to score points by hitting the ball past your opponent's paddle.



Files:-

main.py: This file contains the main code for the Pong game. It orchestrates the interaction between the paddles, ball, and scoreboard. Run this file to start the game.

paddle.py: The code for the paddles is encapsulated in this file. It defines the Paddle class, which handles the movement and display of each player's paddle.

ball.py: This file contains the code for the ball in the game. The Ball class manages the ball's movement, collisions, and rendering.

scoreboard.py: The scoreboard functionality is implemented in this file. It includes the Scoreboard class, which keeps track of each player's score and updates the display accordingly.
